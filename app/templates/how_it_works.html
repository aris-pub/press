Current Features

Content-Addressable Storage

Permanent URLs guaranteed forever. Same content always gets the same address. Built on the same technology Docker uses for software distribution.

Web-Native Rendering

No more PDF pinch-and-zoom. Your papers display beautifully on phones, tablets, and desktops. Native web typography and responsive design.

Accessibility-First Design

Screen reader compatible, keyboard navigation, high contrast support. Research should be accessible to everyone, everywhere.

Basic JavaScript Support

Interactive visualizations, executable code blocks, dynamic content. Your modern authoring tools' features actually work for readers.
