# Changelog

- Human-readable URLs (`scroll.press/2026/glee`) alongside hash URLs

- Slug generation from paper titles at publish time

- Mobile viewport padding on embedded papers
- FAB logo clipping in grouped button layout

### Changed

- Scroll cards and meta tags now use canonical year/slug URLs
